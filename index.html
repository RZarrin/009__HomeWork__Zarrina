Задание: (База пользователей)

создайте пустой объект, цикл от 1 до 10 и при помощи prompt ввести 10 человек (имя и возраст)

Итерация это id пользователя | id не может быть 0. В эту итерацию записываются данные имени и возраста

Потом сделать вывод в console Например:

  1. Пользователь 3: Имя Иван
  2. Возраст 36
  3. Пользователь 4: Имя Елена
  4. Возраст 24

После вывести весь объект в console
